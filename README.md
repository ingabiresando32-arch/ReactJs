# MPUZA DASHBOARD DESIGNED USING REACCTJS & TailwindCss

A **React-based frontend** that simulates a social media-style job posting and networking feed. Users can create posts, view job listings, and interact with posts. Styled with **Tailwind CSS** and enhanced with **Font Awesome icons**.  

---

## 📌 Features

- **Create Post Section**  
  - Post text content and attach images.  
  - Audience selector (`Your Followers` with dropdown).  
  - Display post metrics (e.g., Jobless supported, US $99.99 reward).  

- **Job Cards**  
  - Display company logos, job titles, qualifications, and urgency badges.  
  - Responsive grid layout for mobile, tablet, and desktop.  

- **Post Interaction Section**  
  - Show number of vacant positions and experience requirements.  
  - Countdown timer for job application deadlines.  
  - Action buttons: `Like`, `Comment`, `Share`, `Save`.  

---

## 💻 Technologies Used

- **React** – Component-based UI development.  
- **Tailwind CSS** – Utility-first CSS framework.  
- **Font Awesome** – Icons for posts and actions.
